OXID eShop 6.5.4
================

Veröffentlichungstermin: 27.02.2024

Um eine Sicherheitslücke im Composer zu schließen, installieren Sie OXID eShop 6.5.4.

Aus Sicherheitsgründen erfordert OXID eShop 6.5.4 die Composer-Version 2.7.x.

Weitere Informationen finden Sie unter

* `Composer Version 2.7.0 <https://github.com/composer/composer/releases/tag/2.7.0>`_
* `CVE-2024-24821 <https://nvd.nist.gov/vuln/detail/CVE-2024-24821>`_
